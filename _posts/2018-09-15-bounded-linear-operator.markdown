## 有界線形作用素
### 定義1 (線形作用素)
二つの線形空間$X, Y$に対して写像$T: X \to Y$が以下の線型性を満たすとき，$T$を線形作用素と呼ぶ．

$X, Y$の係数体を$K$とすると，任意の$\alpha, \alpha^{\prime} \in K, x, x^{\prime} \in X$について，

\begin{equation}
T(\alpha x + \alpha^{\prime} x^{\prime}) = \alpha T(x) + \alpha^{\prime} T(x^{\prime}).
\end{equation}


*備考*

定義としては線形写像と同じであるが，想定する線形空間$X, Y$が関数の空間である時，線形作用素と呼ばれる．例えば微分や積分がこれにあたる．

### 例1
係数体$K$上の線形空間$X$に内積$\langle \cdot, \cdot \rangle$を定義し，計量線形空間を構成する．任意に一つ$x \in X$が与えられたときに，$\langle \cdot, x \rangle$は線形作用素となる．これは，内積の双線型性より任意の$x_{1}, x_{2} \in X$について，$$\langle \alpha x_{1} + \alpha^{\prime} x_{2}, x \rangle = \alpha \langle x_{1}, x \rangle + \alpha^{\prime} \langle x_{2}, x \rangle$$が成り立つことから確認できる．

### 命題1

$$0_{X}$$，$$0_{Y}$$をそれぞれ$X, Y$の零元とする．線形作用素$T$に対し，$$T(0_{X}) = 0_{Y}$$が成り立つ．

*証明*

線型性より$$T(0_{X}) = T(0 x) = 0 T(x) = 0_{Y}$$が任意の$x \in X$について成り立つ．$\square$

### 命題2
それぞれノルムの定義された線形空間$X, Y$の線形作用素$T: X \to Y$について，$T$はノルムの順序関係を保存する．つまり，$$\forall x_{1}, x_{2} \in X \bigl( \| x_{1} \|_{X} \leq \| x_{2} \|_{X} \Rightarrow \| T(x_{1}) \|_{Y} \leq \| T(x_{2}) \|_{Y} \bigr)$$が成り立つ．

*証明*

$$\exists x_{1}, x_{2} \in X \bigl(  \| x_{1} \|_{X} \leq \| x_{2} \|_{X} \wedge \| T(x_{1}) \|_{Y} > \| T(x_{2}) \|_{Y} \bigr)$$が成り立つと仮定する．$0 < \alpha \leq 1$を用いて，$x_{1} = \alpha x_{2}$であるとすると，$$\| x_{1} \|_{X} \leq \| x_{2} \|_{X}$$を満たしている．このとき，$$ \| T(x_{1}) \|_{Y} = \| T(\alpha x_{2}) \|_{Y} = \alpha \| T(x_{2}) \|_{Y} \leq \| T(x_{2}) \|_{Y}$$であり，$$\| T(x_{1}) \|_{Y} > \| T(x_{2}) \|_{Y}$$に矛盾する．$\square$

*備考*

後の証明に使われるわけではない．最初使うかと思ったが僕の勘違いで必要なかった．しかしとても線形感があるのでとっとく．

### 定義2 (有界)
線形空間$X, Y$の上にノルムを定義する．線形作用素$T$が有界であるとは，ある$K \geq 0$が存在し，任意の$x \in X$について

$$
\| T(x) \|_{Y} \leq K \| x \|_{X},
$$

が成り立つことである．


### 定義3 (作用素ノルム)
有界線形作用素$T: X \to Y$に対して，そのノルムを

$$
\|T\| \overset{\mathrm{def}}{=} \sup_{x \neq 0_{X}} \frac{\| T(x) \|_{Y}}{\| x \|_{X}},
$$

によって定義する．これを作用素ノルムと呼ぶ．

### 命題3
作用素ノルムについて，以下は同値である．

1. $$\sup_{\|x\|_{X} = 1} \| T(x) \|_{Y}$$,
2. $$\sup_{\|x\|_{X} \leq 1} \| T(x) \|_{Y}$$.

*証明*

まず1を示す．$$\frac{\| T(x) \|_{Y}}{\| x \|_{X}}$$は，ノルムの定義より$$\| \frac{1}{\| x \|_{X} } T(x) \|_{Y}$$．次に$T$の線型性より$$\| T (\frac{x}{\|x\|_{X}})\|_{Y}$$となる．$$\frac{x}{ \| x \|_{X}}$$は$X$の元で，正規化されているためノルムが1となる．$$\frac{x}{\| x \|_{X}}$$を新たに$x$
とおくと，$$\sup_{\|x\|_{X} = 1} \| T(x) \|_{Y}$$である．$\square$

次に2を示す．$x$が$$\| x \|_{X} \leq 1$$を満たす範囲について考える．まず$$\| x \|_{X} = 0$$のとき$$x=0_{X}$$なので命題1より，$$\|T(x)\|_{Y} = 0$$．次に，$$ 0 < \| x \|_{X} \leq 1$$のとき，$$x = \|x\|_{X} \frac{x}{\|x\|_{X}}$$を用いると，

$$
\|T(x)\|_{Y} = \| T(\|x\|_{X} \frac{x}{\|x\|_{X}}) \|_{Y} = \|x\|_{X} \| T(\frac{x}{\|x\|_{X}}) \|_{Y} = \|x\|_{X} \| T(\tilde{x}) \|_{Y}.
$$

ここで，$$\tilde{x} = \frac{x}{\|x\|_{X}}$$であり，正規化しているため$$\| \tilde{x} \|_{X} = 1$$となる．$$ 0 < \| x \|_{X} \leq 1$$であったため，

$$
\|T(x)\|_{Y} = \|x\|_{X} \| T(\tilde{x}) \|_{Y} \leq \| T(\tilde{x}) \|_{Y}
$$

である．$$T(0_{X}) = 0$$より，

$$
\|T(0_{X})\|_{Y} \leq \|T(x)\|_{Y} \leq \| T(\tilde{x}) \|_{Y}
$$
なので，$$\| x \|_{X} \leq 1$$全てについてこれが成り立っていることがわかる．$$\|T(x)\|$$では$x$は$$\| x \|_{X} \leq 1$$を満たしており，$$\| T(\tilde{x}) \|_{Y}$$では$$\| x \|_{X} = 1$$となることに注意して両辺の$$\sup$$をとると，

$$
\sup_{\| x \|_{X} \leq 1} \|T(x)\|_{Y} \leq \sup_{\| x \|_{X} = 1} \| T(\tilde{x}) \|_{Y}.
$$

また，$$\| x \|_{X} \leq 1$$となる$x$の集合は，$$\| x \|_{X} = 1$$となる$x$の集合を含むため，

$$
\sup_{\| x \|_{X} = 1} \| T(\tilde{x}) \|_{Y} \leq \sup_{\| x \|_{X} \leq 1} \|T(x)\|_{Y}

$$

が成り立つ．
従って

$$
\sup_{\| x \|_{X} = 1} \| T(\tilde{x})\|_{Y} = \sup_{\| x \|_{X} \leq 1} \|T(x)\|_{Y},
$$

となる．$\square$

### 命題4

線形作用素$T$について，以下は同値．

1. 線形作用素$T$は有界である．
2. 線形作用素$T$の作用素ノルムが有限である．
3. 線形作用素$T$がリプシッツ連続である．
4. 線形作用素$T$が一様連続である．
5. 線形作用素$T$が連続である．
6. 線形作用素$T$が定義域の零元において連続である．
7. 線形作用素$T$が定義域の1点において連続である．


*証明($1\to2$)*

線形作用素の有界の定義より，ある$K$が存在して$$\frac{\| T(x) \|_{Y}}{\| x \|_{X}} \leq K $$となる．左辺の$\sup$が作用素ノルムの定義であったため，それが有限であることを意味する．$\square$

*証明($2\to3$)*

リプシッツ連続であることを示すため，ある定数$K \geq 0$が存在して，任意の$x_{1}, x_{2} \in X$について

$$
\| T(x_{1}) - T(x_{2}) \|_{Y} \leq K \| x_{1} - x_{2} \|_{X}
$$

が成立することを示せば良い．作用素ノルムが有界であることから，任意の$x \in X$について，$$\frac{\| T(x) \|_{Y}}{\| x \|_{X}} \leq K$$
となる$K \leq 0$が存在する．このとき，$X$はベクトル空間より$x_{1} - x_{2} \in X$であるため，$$\frac{\| T(x_{1} - x_{2}) \|_{Y}}{\| x_{1} - x_{2} \|_{X}} \leq K$$とできる．$T$の線型性より，任意の$x_{1}, x_{2} \in X$について$$\| T(x_{1}) - T(x_{2}) \|_{Y} \leq K \| x_{1} - x_{2} \|_{X}$$．$\square$


*証明($3\to4$)*

任意の$\epsilon>0$について$\delta$を$\frac{\epsilon}{K}$と取れば良い．$\square$

*証明($4\to5$)*

任意の点で成り立っているので各点でも成り立つ．$\square$

*証明($5\to6$)*

任意の点で成り立っているので零元でも成り立つ．$\square$

*証明($6\to7$)*

零元で連続であるとき，$$\forall x \in X \forall \epsilon > 0 \forall \delta >0 \bigl( \|x\|_{X} \leq \delta \Rightarrow \| T(x) \|_{Y} \leq \epsilon \big)$$が成り立ち，$X$は線形空間より$X$の2元について$x_{1} - x_{2} \in X$なので$$\forall x_{1}, x_{2} \in X \forall \epsilon > 0 \forall \delta >0 \bigl( \|x_{1} - x_{2}\|_{X} \leq \delta \Rightarrow \| T(x_{1}) - T(x_{2}) \|_{Y} \leq \epsilon \big)$$が成り立ち，これは$x_{2}$で連続であることを意味する．(正確には6 -> 5 -> 7になっている) $\square$

*証明($7\to1$)*

線形作用素$T$がある1点$x_{0} \in X$で連続であると仮定する．このとき，このとき，任意の$x \in X$について$\epsilon=1$としてある$\delta > 0$が存在して，$$\| x - x_{0}\|_{X} \leq \delta \Rightarrow \| T(x - x_{0})\|_{Y} \leq 1$$が成り立つ．ここで，$$\| x - x_{0}\|_{X} \leq \delta$$が成り立つ$X$の元$x - x_{0}$を$$\Delta$$と書くと，任意の$x \in X$について，$$\bigl\| \delta \frac{x}{\|x\|_{X}}\bigr\|_{X} = \delta$$となり条件を満たす．

$$
\| T(x) \|_{Y} = \delta^{-1} \delta \|x\|_{X}^{-1} \|x\|_{X} \| T(x) \|_{Y} = \delta^{-1} \| x \|_{X} \| T(\delta \frac{x}{\|x\|_{X}}) \| \leq \delta^{-1} \| x \|_{X}.
$$

従って$K = \delta^{-1}$として線形作用素$T$は有界である．$\square$
